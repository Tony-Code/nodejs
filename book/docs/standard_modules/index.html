Standard Modules 标准模块

node 附带了一些模块，这些模板都被编译进了node 二进制文件中，其中大部分将在下文介绍。使用这些模块 最常见的方法是调用“require('name')”并将返回值赋给一个和模块同名的局部变量。 例如：

var sys = require('sys');

同样可以使用其他模块扩展node。详情参看"模块"章节。
